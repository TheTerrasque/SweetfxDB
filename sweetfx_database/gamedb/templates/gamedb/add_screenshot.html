Formats

When uploading, please use the PNG format. If you don't have software capable of saving to PNG, you can use IrfanView.

Comparison image

When using comparison image functionality, try to make the screenshots as similar as possible. One neat trick can be to bind the SweetFX screenshot button to the same as the ingame screenshot button, and thus take two images at (almost) the same time.
